ACDK SQL ODBC
ACDK
Artefaktur Component
Development Kit

With acdk_sql_odbc, you can access all databases providing the ODBC interface.


The acdk_sql_odbc library implements the interfaces of  acdk_sql to access databases via the ODBC interface.


Following Databases should be supported:

Windows all ODBC-Databases, including MS SQL 2000, MS Access, PostgreSQL, Oracle, Informix.

On Linux/Solaris platform:
Oracle 8, mySQL, MiniSQL, PostgreSQL, Text-Files.

acdk_sql_odbc support the  basic SQL Type the  standard queries and operations,  prepared statements invokation of  stored procedures and  database metainfo.

Please refer to  Installation how to use acdk_sql_odbc on Windows and Unix platforms.
