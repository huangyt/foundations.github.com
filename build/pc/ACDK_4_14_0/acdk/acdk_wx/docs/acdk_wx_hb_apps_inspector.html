acdk_wx Inspector
ACDK
Artefaktur Component
Development Kit

Inspector is is type explorer/class browser, a heap/object explorer and a interactive CfgScript console. Inspector is written using a mixture of C++ and CfgScript.

 Start Inspector

The main Inspector application is written in CfgScript. You have to start it with acdkcfgscript interpreter:
acdk/bin/acdkcfgscript[_d] acdk/acdk_core/cfg/csf/lib/inspect/Inspector.csf

 Construction

Inspector is written with a combination of ACDK C++ and CfgScript.
The GUI components ClassTreeCtrl, ObjectTreeCtrl and CfgScriptConsole are written in ACDK C++.
These controls are used and controled in the CfgScript frame acdk/acdk_core/cfg/csf/lib/inspect/Inspector.csf.

 Class View

The Class view ( acdk::wx::inspector::ClassTreeCtrl) is a traditional class browser.
All available type information can be viewed.
Via File / Load DLL/SO additionally ACDK libraries with type information can be loaded.



 Object View


The Object view ( acdk::wx::inspector::ObjectTreeCtrl) display all root objects (Objects, which are not itself member of another Object).



 CfgScript Console

The CfgScript Console ( acdk::wx::inspector::CfgScriptConsole) can invoke the the garbage collector or invoke fragments of CfgScript code.
