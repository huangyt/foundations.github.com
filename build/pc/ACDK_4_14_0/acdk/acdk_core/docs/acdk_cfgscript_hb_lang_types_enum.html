An enum is a collection of integer types. 



Enumeration
: 'enum' TypeName '{' (EnumValues)+ '}'
;

EnumValues
: EnumVal [ ',' EnumValues ]
;

EnumVal
: Identifier [ '='  Expression ]
;

Sample:

enum MyEnumValues
{
  FirstValue, // == 0
  SecondValue, // == 1
  ThirdValue = 10,
  NextValue, // == 11
  LastValue = NextValue + ThirdValue 
}
