AAL (aal)

 Description

aal is an experimental language for scripting with aspect oriented features. Currently it is only a play zone for experimenting with advanced language features, like static and dynamic types. A dynamic compiler should enable the language to enhance the compiler in all aspects at compile time. The dynamic compiler enables to implement aspect oriented techniques into the compiler.
