 Implementation

Freestanding function will translated into classes:
using acdk.lang;
String foo(String s, int i)
{
  return s.substr(i);
} 
foo("Hallo", 1);

// will be translated into
class foo 
{
  public static String operator()(String s, int i)
  {
    return s.substr(i);
  }
}
foo.operator()("Hallo", 1);
