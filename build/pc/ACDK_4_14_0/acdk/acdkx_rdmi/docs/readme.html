 Short Description

  acdkx_rmdi implements a transparent remote invocation for ACDK.

 Dependencies

 ACDK Net

 Namespaces

 Authors

 License

 Description

acdkx_rdmi provides a transparent remoting for all ACDK objects. Currently it is in alpha status. For completion acdkx_rdmi an asynchronous IO framework has to be established. Some  conceptual notes are available.
