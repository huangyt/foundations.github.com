ACDK Java
ACDK
Artefaktur Component
Development Kit

acdk_java makes ACDK Objects accessible to Java and Java classes to C++.
acdk_java integrate an Java virtual machine into acdk.
acdk_java also provides Java compatible serialization of C++ classes.


acdk_java connects ACDK Objects with Java objects without the necessarity to write own Java Native Interface (JNI) code.


Using the standard  ACDK DMI features, acdk_java provides following features:
  • Java can act as an  DMI client for all ACDK DMI server applications.
  • Java can act as a DMI server, makes all Java classes acccessible to all ACDK DMI clients.
    See also  acdk.java.AcdkObject.
  • Embedd a Java Virtual Machine inside ACDK C++ applications.
    See also  acdk_java Manual.
  • With the package  acdk_java_serialization a Java compatible serialization of ACDK C++ Objects is provided.
    See also  acdk_java_serialization Manual.
