ACDK Make
ACDK
Artefaktur Component
Development Kit

acdk_make (amake) contains a build system for ACDK.


acdk_make is target as a make system replacement similar to ant in Java world.
  • Extensionable through Tasks.
  • Support for various compiler/platforms
  • paralized compile Jobs
  • build-in fast dependencies check

Currently acdk_make is alpha software and will change in future.
