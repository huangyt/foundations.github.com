 Usage

 Command Line

acdkperl [acdk-flags] [perl file].
acdkperl want as only argument a perl file to interpret.
If there is no argument, acdkperl read the perl code to interpret from the stdin. 
> acdkperl
print "Hello acdkperl\n";
[Ctrl-D]
prints 
"Hello acdkperl"
on the screen.
