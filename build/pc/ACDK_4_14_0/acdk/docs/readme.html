 Title

Artefaktur Component Development Kit (ACDK)

 Version

Distribution version: 4.14.
Package version: 4.14.0.

 Short Description

ACDK is C++ framework with similar features of Java or C#.

 Documentation

Please refer to the documentation inside  Packages. Only a part of the documentation is enclosed in this package. A complete documentation, including a handbook and API documentation is available at  acdk.sourceforge.net. You can also download the documentation at  Download@sourceforge.

 Download

Current version are available at:  Download@sourceforge.

 Dependencies

none.
Fore some modules you need:
- tcl 8.x
- perl (5.6/5.8)
- python
- unixODBC
- Java 1.3/1.4
- wxWindow/wxWidget
