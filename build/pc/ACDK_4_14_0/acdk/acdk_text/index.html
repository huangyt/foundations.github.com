ACDK Text
ACDK
Artefaktur Component
Development Kit

acdk_text contains classes for text rendering and parsing, including perl like regular expressions.


acdk_text corresponds to the java.text packages.

See also  acdk::text.

Additionally acdk_text contains  acdk::text::RegExp for Perl like regular expression.
